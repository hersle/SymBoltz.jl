Creating extended models

This tutorial shows how to create extended (beyond-О›CDM) models.

As a simple example, we will replace the cosmological constant О› with equation of state $P(t) / \rho(t) = w(t) = -1$ in the pure О›CDM model with the wв‚Ђwв‚ђ-parametrization from Chevallier, Polarski (2000) and Linder (2002) with equation of state

\[w(t) = w_0 + w_a (1 - a(t)).\]

With this equation of state, solving the continuity equation

\[\frac{\mathrm{d} \rho(t)}{\mathrm{d}t} = -3 H(t) \, (ПЃ(t) + P(t)) = -3 H(t) \, \rho(t) \, (1 + w(t))\]

with the ansatz $\rho(t) = \rho(t_0) \, a(t)^m \exp(n (1 - a(t)))$ gives $m = -3 (1 + w_0 + w_a)$, $n = -3 w_a$ and hence

\[\rho(t) = \rho(t_0) \, a(t)^{1 + w_0 + w_a} \exp(w_a (1 - a(t))).\]

Create the symbolic component

SymBoltz.jl represents each component of the Einstein-Boltzmann equations as a partial/incomplete ModelingToolkit.jl ODESystem. These are effectively "chunks" of logically related parameters, variables, equations and initial conditions that are composed together into a full/complete cosmological model. Let us create a component for the wв‚Ђwв‚ђ-parametrization:

    pars = @parameters w0 wa ПЃ0 О©0

    # 2. Create variables
    vars = @variables ПЃ(t) P(t) w(t) Оґ(t) Пѓ(t)

    # 3. Specify background equations (~ means equality in ModelingToolkit)
    eqs0 = [
        w ~ w0 + wa * (1 - g.a) # equation of state
        ПЃ ~ ПЃ0 * g.a^(-3 * (1 + w0 + wa)) * exp(-3 * wa * (1 - g.a)) # energy density
        P ~ w * ПЃ # pressure
    ] .|> SymBoltz.O(Пµ^0) # O(ПµвЃ°) multiplies all equations by 1 (no effect), marking them as background equations

    eqs1 = [
        Оґ ~ 0 # energy overdensity
        Пѓ ~ 0 # shear stress
    ] .|> SymBoltz.O(Пµ^1) # O(Пµ^1) multiplies all equations by Пµ, marking them as perturbation equations
